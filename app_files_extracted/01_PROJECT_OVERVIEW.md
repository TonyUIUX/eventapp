# 📌 Project Overview — KochiEvents App

## App Name Ideas
- **KochiGo** — Short, punchy, local feel
- **EventsKochi** — Clear and searchable
- **Happenin'** — Casual, youthful
- **KochiScene** — Cultural, vibey
> 🏆 Recommended: **KochiGo** — easy to brand, remember, and scale

---

## Vision
A hyper-local event discovery app for Kochi, Kerala. No clutter, no login, no noise — just what's happening today and this weekend in your city.

## Problem We're Solving
People in Kochi miss out on events (comedy nights, tech meetups, workshops, gigs) because there's no single trusted source. WhatsApp forwards and Instagram stories are the only channels — both are unreliable and ephemeral.

## Target Users
- Age 18–35, Kochi-based or visiting
- Students, young professionals, expats
- People looking for weekend plans
- Event-goers who hate complicated apps

---

## MVP Scope (v1.0)

### ✅ IN SCOPE
- Browse events (Today / This Weekend toggle)
- Filter by category
- View event details
- Save/bookmark events locally
- No login required

### ❌ OUT OF SCOPE (Do NOT build)
- User login / authentication
- Social features (comments, sharing, RSVP)
- Ticketing / payment
- Push notifications
- Maps integration beyond a clickable link
- User-generated content
- Reviews or ratings
- Chat

---

## Success Metrics for MVP
- App loads in < 2 seconds
- Events display correctly from Firestore
- Filter works instantly (client-side)
- Saved events persist across app restarts
- Zero crashes on Android 8.0+

---

## Business Context
- Budget: Lean MVP ($100 equivalent)
- Timeline: Launch-ready in 30 days
- Platform: Android only (Play Store)
- Monetization: None in v1. Future: promoted listings, sponsorships
- Admin: Manual Firestore data entry by team (no CMS needed for v1)

---

## Scale Plan (Post-MVP — Do NOT build now)
- iOS version
- Admin dashboard / CMS for event organizers
- Push notifications for saved events
- Ticket booking via Meesho/BookMyShow API
- City expansion (Trivandrum, Thrissur)
